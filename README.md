# BossIpSwitcher

Windows 10/11 单文件 IP 快速切换工具。

- `Ctrl + Windows + Alt`：固定网络与自定义网络之间切换。
- `Ctrl + Alt + F10`：输入密码后打开设置。
- 固定模式始终使用 `20.65.32.199 / 255.255.255.0 / 20.65.32.254`。
- 备用模式打开记事本；切回固定模式时关闭由程序打开的记事本。
- 程序以管理员权限运行，并定时检测当前 IPv4 地址，发现变化后自动恢复。
- 守护服务采用连续异常确认和重配冷却，避免断网、休眠或网关短暂消失时反复重置 TCP/IP。

## 关于自动分配网段中的固定 IP

本机手动设置静态 IP 不能阻止 DHCP 服务器把同一地址租给其他设备。长期独占某个地址必须由网络管理员在 DHCP 服务器或路由器上按本机网卡 MAC 地址建立“地址保留”，或者把该地址从 DHCP 地址池中排除。不要仅依靠本程序抢占地址，否则可能产生 IP 冲突。

注意：拥有 Windows 管理员权限的用户始终能停止或卸载任何本机程序，因此无法做到绝对不可篡改。该程序提供的是运行期间的快速自动恢复，不是操作系统级安全边界。
